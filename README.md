## 🚀 Overview

**VideoHub** is a full-stack video hosting platform designed for high-performance streaming. It allows users to upload "Reels" and manage their feed with a seamless, responsive UI that adapts to user preferences.

## ✨ Features

- **🌓 Dynamic Theme Toggle**: Perfectly synced Dark and Light modes using `next-themes`.
- **⚡ Blazing Fast Video**: Powered by ImageKit SDK for real-time video transformations.
- **🔒 Authentication**: Secure user login and registration powered by NextAuth.js.
- **🛠️ Full CRUD**: Authenticated users can Edit and Delete their reels directly from the UI.
- **📱 Responsive Design**: A mobile-first grid layout that fits every video into a perfect 16:9 frame.
